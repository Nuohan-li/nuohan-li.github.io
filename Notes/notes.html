MY WEB DEVELOPMENT NOTES

userful tags HTML

center: puts everything in between the opening tag and closing tag in the middle

hr: tag that adds a horizontal line

em: emphasizes the text and makes the texts italic

strong: make texts bold

ul: unordered list - bullet point

ol: ordered list - numbers

  1. type="i" numbers in Roman numerals
  2. start="2" numbers start from 2

li: list item in lists

img src="" : puts a image


table : creates a table, no styles at all by default when created

thead : creates a header for the table, will be bolded automatically

tbody : creates a body for the table

tfoot : creates a footer for the table

tr : table row

td : item in each cell


form: create a form, attributes include action, method, enctype etc.

label: create a label for the form

input: a control to get data from user, there are many types of input such as color picker, text box, uploading file etc. see input

height

text color: user color: to set the text to desired color, see text color



Path

. means the current path level (so if you're on index.aspx and you reference ./style.css then the latter would have to be in the same folder as the former)

.. means one path level up (so if you're on /somefolder/index.aspx and you reference ../style.css then the latter would have to be in the parent folder of someFolder)

/ means the root level (so /style.css is the same as http://www.mysite.com/style.css)



miscellaneous

meta charset="" : tells the browser to encode all the texts in the file to the chosen encoding system
