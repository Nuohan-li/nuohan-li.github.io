Tic Tac Toe

The classic game that every one has played. This is also my very first project done in Python. Pygame library is used to create this game. The game takes two players, each player take turn to place a cross or a circle on the available spot, until one player got three of her/his marks in a row (coloumn or diagnal).

Snake Game

Clone of the class Nokia snake game, the game that I enjoyed so much as a child. This game is also written in Python using Pygame library. Player controls the snake using the arrow keys to eat as much fruits as possible, while avoiding hitting on the wall or the body of the snake.

DRandomBot

A discord bot that is capable of doing many things, such as telling a dad joke, getting cat pictures, finding memes, send anime pictures and telling a city's weather. The bot is programmed in Python, using discord.py library.

All of the above functionalities are suggested by my friends, should you have any other suggestions, please do let me know using the contact me function in top navigation bar.
